# 📖 Panduan Upgrade macOS: Yosemite → Monterey

Panduan langkah demi langkah untuk upgrade macOS dari **Yosemite (10.10)** ke **Monterey (12)** pada MacBook Pro Retina 13-inch Early 2015 (A1502 / EMC 2835). Ditulis dalam **Bahasa Indonesia** untuk pengguna non-teknis.

## Tentang

MacBook Pro Early 2015 yang masih menjalankan macOS Yosemite sudah tidak menerima update keamanan sejak lama. Panduan ini membantu Anda melakukan upgrade ke **macOS Monterey** — versi resmi tertinggi yang didukung Apple untuk model ini — dengan aman dan terstruktur.

## Apa yang Dibahas

- ✅ Pengecekan kompatibilitas perangkat
- ✅ Persiapan ruang penyimpanan
- ✅ Backup data dengan Time Machine atau manual
- ✅ Jalur upgrade bertahap (Yosemite → El Capitan → Monterey)
- ✅ Proses instalasi step-by-step
- ✅ Setup pasca-instalasi
- ✅ Troubleshooting & langkah darurat
